# 07 场景化应急预案

定位：把通用流程转成高频事件的可执行 Runbook。

## Runbook 待办

- [ ] 账号失陷、异常登录、MFA 疲劳与凭据泄露。
- [ ] 钓鱼、恶意附件、恶意链接与商业邮件欺诈。
- [ ] 终端恶意代码、勒索软件和横向传播。
- [ ] Web 入侵、WebShell、漏洞利用与 API 滥用。
- [ ] 数据泄露、数据外传、对象存储暴露与第三方泄露。
- [ ] DDoS、挖矿、资源滥用与可用性事件。
- [ ] 云账号接管、容器/Kubernetes 安全事件与供应链事件。

## 每份预案必须包含

- [ ] 触发信号、定级标准、首 15 分钟动作、证据、遏制、恢复、沟通和复盘。
